# Changelog

## 14.06.2026

- Реализована работа с несколькими треками — пакетный импорт, список файлов, цветные полилинии на карте, массовый экспорт (до 3 файлов по отдельности, 4+ в ZIP с сохранением `originalFileName`)
- Добавлены фильтры пост-обработки треков — шумоподавление (скользящее среднее), оптимизация RDP (`simplify-js`), сглаживание Чайкина; глобальные настройки с debounce 175 мс ([подробно в спецификации](docs/spec.md#тз-модуль-пост-обработки-и-сглаживания-gpx-треков), [алгоритмы — algos.md](docs/algos.md))
- Архитектура `TrackStore` переведена на коллекцию `TrackItem` с разделением `rawPoints` / `points` (computed)
- Добавлено количество точек в блок метрик
- Имя файла при экспорте формируется из времени трека: `Jun_12_2026_at_20:31.gpx`

## 13.06.2026

- MVP: импорт GPX (drag-and-drop), карта OSM/Leaflet, метрики, профиль высот, экспорт GPX
- Двухпанельный интерфейс с адаптивным сайдбаром (гамбургер-меню на мобильных)
